# 三份样本案例结构化结果（可直接写进技术文档/PPT）

## 1. 崔某劳务合同纠纷二审民事判决书

- 案由：劳务合同纠纷
- 原告/申请人：崔某
- 被告：北京某公司、张某
- 诉请：北京某公司、张某支付崔某劳务费 23127 元
- 争议焦点：
  - 北京某公司是否应就本案农民工主张的工资承担清偿责任
  - 崔某主张的欠付工资是否属实
- 法院认定事实：
  - 北京某公司系案涉工程承包方，将施工工作交由张某，允许张某自行雇佣、管理工人施工，并直接与张某结算工程款
  - 崔某提交工资表、工资发放表及微信聊天记录，张某确认欠薪事实和金额
  - 北京某公司未能提交足以推翻工资表真实性的证据
- 裁判结果：驳回上诉，维持原判；张某、北京某公司共同支付劳务费用 23127 元
- 适用法条：
  - 《中华人民共和国民法典》第五百零九条
  - 《中华人民共和国民法典》第五百七十九条
  - 《保障农民工工资支付条例》第十九条、第三十六条
  - 《中华人民共和国民事诉讼法》第六十七条、第一百四十七条、第一百七十七条第一款第一项
  - 《最高人民法院关于适用<中华人民共和国民事诉讼法>的解释》第九十条

## 3. 张某云与张某森离婚纠纷支持起诉案（检例第126号）

- 案由：离婚纠纷（支持起诉）
- 原告/申请人：张某云
- 被告：张某森
- 诉请：请求离婚；请求检察机关支持起诉维权
- 争议焦点：
  - 张某云是否长期遭受家庭暴力并导致夫妻感情确已破裂
  - 检察机关是否可以依申请支持家庭暴力受害人提起离婚诉讼
- 法院/检察机关认定事实：
  - 张某云在第一次离婚诉讼后继续遭受张某森殴打、恐吓
  - 检察机关通过报案材料、出警记录、伤情照片、微信聊天记录等调查核实，确认其长期遭受家庭暴力
  - 检察机关对其开展心理疏导和反家庭暴力法宣讲，并支持其再次起诉
- 裁判结果：武邑县法院一审准予离婚；衡水市中级人民法院调解确认双方离婚，并对子女抚养、财产分割等达成协议
- 适用法条：
  - 《中华人民共和国民事诉讼法》第十五条
  - 《反家庭暴力法》第四条
